## Työn aihe

### Kalamiehen saalispäiväkirja 

Saalispäiväkirja on yksinkertainen tietokantasovellus, jonka avulla kalastajat voivat pitää kirjaa niin saaliistaan kuin pyyntivälineistäänkin.
